## Get started with Lexa in three steps

Set up Lexa LLM and start building AI-powered applications in minutes.

### Step 1: Get your API credentials

<AccordionGroup>
  <Accordion icon="key" title="Create your Lexa account">
    Visit [lexa.chat](https://lexa.chat) and create your account to access the Lexa API.
    
    <Tip>
    Lexa offers a generous free tier to get you started with AI development.
    </Tip>
  </Accordion>
  <Accordion icon="copy" title="Generate your API key">
    1. Navigate to your [API settings](https://lexa.chat/settings/api)
    2. Click "Generate New API Key"
    3. Copy your API key and store it securely
    
    <Warning>
    Never share your API key publicly or commit it to version control. Use environment variables instead.
    </Warning>
  </Accordion>
</AccordionGroup>

### Step 2: Make your first API call

<AccordionGroup>
  <Accordion icon="code" title="Using cURL">
    Test Lexa with a simple cURL request:
    
    ```bash
    curl -X POST https://api.lexa.chat/v1/chat/completions \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "lexa-1",
        "messages": [
          {"role": "user", "content": "Hello! What can you help me with?"}
        ]
      }'
    ```
  </Accordion>
  <Accordion icon="javascript" title="Using JavaScript">
    Install the Lexa SDK and make your first call:
    
    ```bash
    npm install @lexa/sdk
    ```
    
    ```javascript
    import { Lexa } from '@lexa/sdk';
    
    const lexa = new Lexa({
      apiKey: process.env.LEXA_API_KEY
    });
    
    const response = await lexa.chat.completions.create({
      model: 'lexa-1',
      messages: [
        { role: 'user', content: 'Hello! What can you help me with?' }
      ]
    });
    
    console.log(response.choices[0].message.content);
    ```
</Accordion>
  <Accordion icon="python" title="Using Python">
    Use the Python SDK for easy integration:
    
    ```bash
    pip install lexa
    ```
    
    ```python
    import os
    from lexa import Lexa
    
    client = Lexa(api_key=os.getenv('LEXA_API_KEY'))
    
    response = client.chat.completions.create(
        model="lexa-1",
        messages=[
            {"role": "user", "content": "Hello! What can you help me with?"}
        ]
    )
    
    print(response.choices[0].message.content)
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Explore advanced features

<Accordion icon="rocket" title="Try different models">
  Lexa offers multiple models optimized for different use cases:
  
  - **lexa-1**: General purpose, balanced performance
  - **lexa-1-fast**: Optimized for speed and efficiency
  - **lexa-1-code**: Specialized for code generation and analysis
  
  <Tip>
  Start with lexa-1 for most use cases, then experiment with specialized models.
  </Tip>
</Accordion>
